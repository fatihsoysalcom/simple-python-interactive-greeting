# Simple Python Interactive Greeting

This example demonstrates a very basic interactive Python script, suitable for beginners. It takes user input for their name and favorite programming language, then prints a personalized greeting. This script illustrates fundamental Python concepts like print statements, input functions, and f-strings for string formatting, which are typically covered when first learning Python and setting up a development environment like VS Code.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Open your terminal or command prompt.
3. Navigate to the directory where you saved the file and run it using `python main.py`.
